Privacy Policy

Last updated: February 2, 2026

Overview

This website (sullivan76.com) is operated by Ryan Sullivan for personal use. This privacy policy explains how data from SMA solar monitoring API is used.

Data Collection

We collect solar production data from SMA APIs solely for personal home automation purposes, including:

  • Real-time solar power generation data
  • Historical energy production statistics
  • System performance metrics

Data Usage

The collected data is used exclusively for:

  • Automating electric vehicle charging based on solar production
  • Personal energy monitoring and optimization
  • Home automation decisions

Data Sharing

We do not share, sell, or distribute any data collected from SMA APIs to third parties. All data remains private and is used solely for personal home automation purposes.

Data Storage

Data is temporarily stored locally for operational purposes and is not transmitted to external services except as required for the automated systems to function.

Security

We implement appropriate security measures to protect access to API credentials and collected data.

Contact

For questions about this privacy policy or data handling, please contact Ryan Sullivan at rasullivan@ucsd.edu

Changes to This Policy

This privacy policy may be updated as needed. The "last updated" date at the top of this page reflects the most recent revision.
